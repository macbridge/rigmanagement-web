📖 使い方マニュアル

1. アプリの概要

アジングリグ在庫管理は、アジングで使うリグ・消耗品の在庫をカテゴリ別に管理するアプリです。画面下部のタブで5つの画面を切り替えて使用します。

  • ホーム：全カテゴリの在庫一覧と在庫不足アラート
  • ジグ単：ワーム・ジグヘッド・フックの管理
  • 遠投リグ：フロート・TGシンカー・スプリット・Mキャロ・イージーフロートワイヤーの管理
  • ライン：ライン・リーダーの管理
  • 設定：データ一括削除など

2. ホーム画面

全カテゴリのアイテムをまとめて確認できるダッシュボードです。

在庫一覧モード

画面上部のセグメントで「在庫一覧」を選ぶと、在庫のあるアイテムがカテゴリ別（ワーム／ジグヘッド／フック／遠投リグ／ライン／リーダー）にまとめて表示されます。セクションのタイトルには在庫件数が表示されます。

在庫不足モード

「在庫不足」を選ぶと、アラート閾値以下になったアイテムのみが表示されます。すべての在庫が十分な場合は「在庫不足なし ✓」が表示されます。

スワイプ操作

  • 右スワイプ（緑）：在庫を +1 します
  • 左スワイプ（オレンジ）：在庫を -1 します（0以下にはなりません）

ホーム画面の一覧からは削除操作はできません。削除は各カテゴリ画面から行います。

詳細画面へ移動

アイテムの行をタップすると詳細画面が開き、登録情報の確認・使用の記録・編集・削除ができます。

3. ジグ単タブ

ワーム・ジグヘッド・フックを管理します。画面上部のセグメントでカテゴリを切り替えます。

一覧の見方

  • 写真が登録されているアイテムはサムネイルが左側に表示されます
  • 在庫が0のアイテムはリストの末尾に薄く表示されます
  • 在庫アラート対象（閾値以下）のアイテムは商品名が赤く表示されます

検索

画面上部の検索バーに入力すると、商品名・メーカー名・カラー（ワーム）・サイズ（フック）で絞り込まれます。

ソート

右上の並び替えアイコンをタップしてソート順を選択します。

  • 在庫少ない順（デフォルト）
  • メーカー順
  • 登録日順

アイテムを追加する

1
右上の「＋」ボタンをタップして追加画面を開きます
2
メーカー・商品名をプリセットから選択するか、直接入力します
3
サイズ・カラー（ワーム）、フックサイズ・重さ（ジグヘッド）、サイズ（フック）などを入力します
4
「追加数」に初期在庫数を入力します（入力した数が在庫に加算されます）
5
必要に応じて写真を選択し、「保存」をタップします

スワイプ操作

  • 右スワイプ（緑 +1）：在庫を 1 増やします
  • 左スワイプ（オレンジ -1）：在庫を 1 減らします
  • 左スワイプ（赤 削除）：アイテムを削除します（確認なし）

ワームの登録項目

  • メーカー / 商品名 / サイズ / カラー
  • 在庫追加数 / アラート閾値 / 写真

ジグヘッドの登録項目

  • メーカー / 商品名 / フックサイズ / 重さ（g）
  • 在庫追加数 / アラート閾値 / 写真

フックの登録項目

  • メーカー / 商品名 / サイズ
  • 在庫追加数 / アラート閾値 / 写真

4. 遠投リグタブ

フロート・TGシンカー・スプリット・Mキャロ・イージーフロートワイヤーを管理します。画面上部のセグメントで種類を切り替えます。

遠投リグの種類

  • フロート：シャローフリーク等のフロート類
  • TGシンカー：フリリグシンカーTG等のタングステンシンカー
  • スプリット：スプリットシンカー
  • Mキャロ：Mキャロ ver.II・ミニMキャロ等
  • EFW：レンジシュートイージーフロートワイヤー等

登録項目

  • 種類（フロート / TGシンカー / スプリット / Mキャロ / EFW）
  • メーカー / 商品名 / サブタイプ / 重さ（g）
  • 在庫追加数 / アラート閾値 / 写真

ソート順

遠投リグは「重さ順」「在庫少ない順」「メーカー順」「登録日順」でソートできます。

5. ラインタブ（ライン・リーダー）

ラインとリーダーをまとめて管理します。画面上部のセグメントで「ライン」「リーダー」を切り替えます。

ラインの登録項目

  • メーカー / 商品名 / 素材（エステル / PE / フロロカーボン / ナイロン）
  • 号数 / lb / カラー
  • 在庫追加数 / アラート閾値 / 写真

リーダーの登録項目

  • メーカー / 商品名 / 素材（フロロカーボン / ナイロン / PE）
  • 号数 / lb
  • 在庫追加数 / アラート閾値 / 写真

プリセット

よく使われるラインとリーダーがプリセットとして登録されており、商品名を選ぶとメーカー・素材が自動入力されます。リーダーは号数を選ぶとlbも自動入力されます。

6. 在庫を増やす（新規購入の記録）

アイテムを購入したときは、編集画面の「追加数」で在庫を増やします。

1
一覧からアイテムをタップして詳細画面を開くか、リストを右スワイプして +1 します
2
詳細画面右上の「編集」ボタンをタップします
3
「在庫」セクションのステッパーで追加数を入力します。「追加後の在庫」にリアルタイムで計算結果が表示されます
4
「保存」をタップすると、追加数が現在の在庫に加算されます

※ 新規登録時は「追加数」が初期在庫として登録されます。

7. 使用を記録する（在庫を減らす）

釣行後に使ったアイテムの在庫を記録します。

1
一覧からアイテムをタップして詳細画面を開きます
2
左上のナビゲーションバーにある「使用を記録」ボタン（マイナスアイコン）をタップします
3
「使用数」ステッパーで使った数を入力します。「記録後の在庫」にリアルタイムで計算結果が表示されます
4
「記録する」をタップすると在庫から使用数が差し引かれます

※ 在庫が0のときは「使用を記録」ボタンが無効になります。

※ ホーム画面・一覧画面の左スワイプ（-1）でも在庫を減らせます。

8. 写真の登録

各アイテムに1枚の写真を登録できます。一覧表示時にサムネイルで確認でき、詳細画面では大きく表示されます。

写真を追加する

新規追加・編集画面の「写真」セクションで「写真を選択」をタップし、フォトライブラリから写真を選択します。

写真を変更・削除する

編集画面で再度「写真を選択」すると写真が上書きされます。写真の下の「写真を削除」をタップすると写真が削除されます。

ファイルサイズについて

登録時に自動で約50KB以下にリサイズされます。元の写真ファイルには影響しません。

9. 在庫アラート

アイテムごとに「アラート閾値」を設定できます。在庫数がアラート閾値以下になると、一覧の商品名が赤く表示され、ホーム画面の「在庫不足」リストに掲載されます。

設定方法

新規追加・編集画面の「アラート閾値」欄に数値を入力します（0に設定するとアラートは発生しません）。

おすすめの使い方

  • ワーム：3〜5 に設定して釣行前の補充忘れを防ぐ
  • ジグヘッド・フック：2〜3 に設定しておくと安心
  • ライン：1 に設定してスプール切れを防ぐ

10. 設定

データ一括削除

「データ一括削除」をタップすると、カテゴリを選んで一括削除できます。トグルで削除するカテゴリを選択し、「削除を実行」をタップします。削除したデータは復元できませんのでご注意ください。

プリセットについて

本アプリは初回起動時に、代表的なアジングタックルのカタログデータ（在庫0・プリセット）を自動登録します。これらは在庫一覧には表示されません。新規追加画面でプリセットから商品を選ぶと、メーカー名などが自動入力されます。
